# LiveDataBus
使用 LiveData 去实现 EventBus，代码非常短，可以直接拷贝到项目中使用。

支持：
- 粘性消息（observeSticky）
- 非粘性消息（observe）
- 不受组件生命周期影响的订阅（observeForever）
